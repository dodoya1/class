# クラスとは

クラスとは、設計書のようなもの。例えば、プロフィールを作る際の設計書について考える。

プロフィールを作る際には、名前、国籍、名前などの情報が必要だと考えられる。

そこで、名前、国籍、名前を入力する入力欄を作成する。それがクラスである。

その後で、その入力欄に具体的に情報を入力することでオーダーメイドのプロフィールを簡単に作ることができる。
